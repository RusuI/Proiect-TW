General description

Product functions:

WhaF comes with a suite of features such as:

  • Search for recipes according to a list of ingredients and other criteria;
  • The possibility of creating an account, which offers additional benefits to the user;
  • Saving some recipes in the favorites section;
  • Viewing a section of popular recipes / ingredients as well as a section of avoided ingredients;
  • Possibility to ask questions regarding the questions related to the application;
  • Posibilitatea adaugarii unei retete proprii;
  • Possibility to add your own recipe;

User classes and features

Users are divided into two main classes:

  1. Users without an account
    • These users do not want to create an account because their main purpose is only to search for recipes based on ingredients;
    • From the point of view of using the application, the frequency of users on the site is low;
    • The results and preferences of the last search will be saved at the browser level;
  2. Users who have an account - accessing the account on the web application offers benefits that can differentiate users into other classes:
    • the class of users who are just looking for recipes, who can save their networks in a favorites section, can recreate networks posted by other users and can post personal networks;
    • the class of experienced users in the field, such as professional chefs, who can promote their networks and possibly a business or a personal blog through our platform.

Operating envinronment

The application is in the first phase intended for use in the browser, on a Windows operating system, on any version.

Requirements for external interface

User Interface

Home Page

The HomePage is the user's first interaction with the "WhaF" application. The display contains a navigation bar (which will later be present on the rest of the associated pages) with the following features:

  • HELP – button that leads the user to the "Frequently Asked Questions" page
  • SEE MOST POPULAR – button that leads the user to the page with the same name
  • GET STARTED – the button that gives the user the main functionality of the application: searching for recipes based on ingredients
  • JOIN US – the button that leads the user to the Sign In / Sign Up form for authentication within the application, respectively, the registration to it
  • PICK INGREDIENTS is an interactive button that has the same functionality as the "GET STARTED" option. The latter is present several times on the Home page, to give the user the ability to search of ingredients from several parts

Help Page - Frequently asked questions

  • In the FAQ section, the user can view the questions and answers corresponding to the existing ones.
  • If the user is logged into their account,then they can upload his own questions by filling in a field with his name and another with the question itself.

See Most Popular

This page can be accessed by any user. Here it is possible to:

  • See popular recipes / ingredients and unpopular ingredients
  • Download the ranking in HTML, CSV, and JSON format
  • Add a recipe to favorites:

Search

To search for a recipe a user can:

  • Select the ingredients you want (Wanted) and not (Unwanted) from the Side Bar. For a more complex filtration he can select the level of difficulty, the preparation time, the cooking time and the number of servings.
  • Remove some previously added criteria by clicking x on the tag.

In the main part there are recipes that match the search, the user can:

  • Add a recipe to favorites:
  • Add a photo with that recipe,marking it as prepared:
  • Access the recipe page (Read more)

Recipe

When the user accesses the profile page of a recipe, he has access to the following functionalities:

  • You can mark a recipe as "saved" for later access to it from the favorite recipes section
  • Sectiunea de review care indica si numarul de poze incarcare cu rezultatul retetei

In aceasta sectiune, utilizatorul logat poate incarca o poza cu rezultatul final al retetei, introducand numele. Acesta selecteaza poza de pe propriul dispozitiv.

Log In and Register

  • It is accessible to the user via the JOIN US button in the navigation bar and contains a login form if the user already has an account created, or a registration form otherwise.
  • The "Log In" button redirects the user to the home page
  • The "Sign Up" button redirects the user to the login page to get into their account.

Profile

  • Once the user accesses his account and implicitly the profile, he will also have the possibility to add his own recipe based on a form indicated by the ADD RECIPE button.
  • PROFILE SETTINGS - the button gives the user the possibility to change the settings regarding the information about his account.
  • LOG OUT – logging out of the application
  • SEE FAVORITES – offers the possibility to view a list of recipes saved to favorites
  • SEE RECIPES - offers the possibility to access a section in which the recipes that the user uploads in the application will be saved

Add Recipe

  • A user can upload his own recipe, filling in the fields with different information such as name, description, cooking time, etc.
  • It is also mandatory to upload one or more pictures with the product.
  • The list of ingredients and preparation steps will be made by adding them via the Add To List button.
  • The recipe will be uploaded via the Post Now button.

Favorites

Pagina de favorite :

  • Strictly represents a list of recipes that the user has saved for later viewing, with the ability to remove them from this list at any time.
  • The "Read More" option will lead him to the recipe page.

My Recipes

This page can be accessed by a logged in user. Here a user can:

  • Similar to the Favorites page, a series of recipes that the user uploads within the application and that become available to other users.
  • The "See Recipe" button will contain a link to the recipe page.

Account Settings

This page can be accessed by a logged in user. Here a user can change their profile settings, such as name, description, email and profile picture. To analyze the operation you must click on the button Upload.
